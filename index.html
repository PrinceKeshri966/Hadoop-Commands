Hadoop Commands Guide (Windows & Linux)

Check Java Version

java -version

Displays the installed Java version.

Format NameNode

hdfs namenode -format

Formats the Hadoop NameNode.

Start Hadoop Services

Windows: start-all.cmd
Linux: ./start-all.sh

Starts all Hadoop daemons.

Stop Hadoop Services

Windows: stop-all.cmd
Linux: ./stop-all.sh

Stops all Hadoop daemons.

Check Running Hadoop Processes

Windows: jps
Linux: jps

Lists running Java processes, useful for checking if Hadoop daemons are active.

Create a Directory in HDFS

hdfs dfs -mkdir /data1

Creates a directory in HDFS.

List Files in HDFS

hdfs dfs -ls /

Lists all files and directories in the HDFS root.

View File Contents in HDFS

hdfs dfs -cat /data1/t1.txt

Displays the content of a file stored in HDFS.

Copy File from HDFS to Local

# Change directory paths according to your system
Windows: hdfs dfs -get /data1/t1.txt C:\your\local\path\t1.txt
Linux: hdfs dfs -get /data1/t1.txt ~/your/local/path/t1.txt

Copies a file from HDFS to the local system.

Copy File from Local to HDFS

# Change directory paths according to your system
Windows: hdfs dfs -put C:\your\local\path\t1.txt /data1/
Linux: hdfs dfs -put ~/your/local/path/t1.txt /data1/

Uploads a file from the local system to HDFS.

Copy File Within HDFS

hdfs dfs -cp /KMP8/T1.TXT /data

Copies a file from one HDFS directory to another.

Get Help for HDFS Commands

hdfs dfs -help

Displays help information for HDFS commands.

Check Disk Usage in HDFS

hdfs dfs -du /directory_name

Displays disk usage statistics for a specific directory in HDFS.
